________Steps________

Create local directory

Command "git status"

Command "Git init" to intialize git directory

Command "git status"

To create global (global=local, Local can be different.) email address/user Command "git config --global user.email" / "git congig --global user.name"

Make a index.html file for testing : add context using command : notepad index.html

Execute command : "git status" it will ask you to "git add" to add in 'staging area' to know the git and add into 'git commit' in git repository as file just saved in working directory

To add in staging area command : "git add 'file name'" e.g "git add index.html" for multiple files to add "git add ."

Optional to remove "git rm --cached 'file name'"

To add in repository command "git commit -m 'First Commit for index file'"

To know the changes user log : "git log"

"git log --oneline" To know the git id: It will show you your all commits (only to see first 3 lines then use "git log --oneline -3".

GitHub is a code hosting platform for version control and collaboration, It let you and others to work together on project to anywhere

git remote add origin git@github.com:abhishekpawar-asp/Test.git
